# JTR-114

- Date: 2026-03-02
- Scope: align ticket detail mobile experience with action-first and collapsible layout

## What changed

- Added mobile-only ticket detail header:
  - back button (`Back to tickets`),
  - compact title + status badge.
- Improved mobile action row:
  - `Start Job` triggers status transition,
  - `Navigate` opens map search by active location address,
  - `Call Customer` uses `tel:` flow when a phone number is found in ticket description.
- Added mobile collapsible sections:
  - `Details`,
  - `Description`,
  - `Comments` (timeline + comment form controls).
- Desktop card layout for these sections is preserved (`md` and above) and hidden on mobile.
