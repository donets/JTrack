# Local Ports 3010/3011/5433

- Date: 2026-02-28
- Scope: move local web/api/postgres host ports to avoid collisions with other containers

## What changed

- Updated dockerized local runtime ports:
  - web: `3010`
  - api: `3011`
  - postgres: `5433` (host) -> `5432` (container)
- Updated compose runtime environment for cross-origin wiring:
  - `WEB_ORIGIN=http://localhost:3010`
  - `NUXT_PUBLIC_API_BASE=http://localhost:3011`
- Updated Docker image metadata:
  - `docker/Dockerfile.web` now `EXPOSE 3010`
  - `docker/Dockerfile.api` now `EXPOSE 3011`
- Updated local defaults/documentation:
  - `docker/docker-compose.yml` postgres port mapping changed to `5433:5432`
  - `.env.example` local `DATABASE_URL` host port changed to `5433`
  - `apps/api/src/main.ts` fallback local API port default changed to `3011`
  - `apps/web/nuxt.config.ts` fallback `apiBase` changed to `http://localhost:3011`
